Description

This course covers the theory and practice of Machine Learning and its fundamental applications in the field of Financial Engineering. Supervised, unsupervised, and reinforcement learning paradigms are discussed. The purpose is to become knowledgeable enough about the field to pursue individual topics further.  In addition, the topics will include practical implementation of the techniques in Python on financial data or other sample data when financial data not available. Students are strongly encouraged to peruse, research and present recent developments on machine learning in Finance as part of their final project.

Objectives

The course introduces the student to the technical and algorithmic aspects of Machine Learning, with equal emphasis on understanding the principles (the why) as well as the methods (the how).

The problem domain is a variety of financial applications, such as model construction and calibration, price prediction, capital allocation, risk management, and trading strategy optimization.

At the end of the course, students will be in command of the basic tools and main financial concepts, to be able to successfully pass the quantitative interview process and start a career in the financial industry.

  • Lecture attendance and participation in the in-class exercises

  • Homework and projects, to be submitted within the stated deadlines

  • The class will use Jupyter Notebooks and all students are expected to run code and do homeworks in that environment.

Prerequisites

  • Intermediate linear algebra, calculus, and statistics
  • Basic knowledge of capital markets and financial instruments (bonds, equities, foreign exchange, simple options, market microstructure)
  • Intermediate Python programming (basic data structures and algorithms, functions and classes, modules and packages)

Readings

There is no main textbook, but the students get reading assignments from a variety of books (see references at the end). A significant portion of the course content overlaps A. Géron's "Hands-on machine learning" text, and students are strongly recommended to obtain a copy.

See the Syllabus for the reading for each week.

Software

The main development environment is the Anaconda Python distribution, which is freely available. The distribution contains all the necessary packages and code development tools (IPython, Spyder, Jupyter). Public and commercial data sets are used for practice.

Course Outline: Material will not necessarily be taught in this order and some topics are subject
to be added/removed.
 
Part I: Classical Machine Learning in Finance
 Introduction to machine learning in finance and Python
 End to End Machine Learning using Scikit-Learn 
 Mathematical Foundations of Machine Learning
 Supervised Learning (with applications in Finance): 
a.     KNN
b.    Regression
c. Kalman filtering
d. Classification and Categorization
e. Support Vector Machines
f. Decision Trees and Random Forests
 Unsupervised Learning: Agglomerative clustering for portfolio construction
 Feature Selection: Constrained regression and VaR hedging
 Classification with Imbalanced data sets
 
Part II: Neural Networks and Deep Learning
 Working with Tensor Flow and Keras
 Artificial Neural Networks
 Training Deep Neural Networks (Vanishing/exploding gradients, Regularization
Techniques, fine tuning hyperparameters)
 Convolution Neural Networks
 Recurrent Networks (GRU, LSTM, Deep RNN, Natural Language Processing)
 AutoEncoders and GANs
 
Reinforcement Learning (time permitting)
